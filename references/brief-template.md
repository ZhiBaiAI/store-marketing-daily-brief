# Brief Template

Use concise, merchant-facing language. Make every section actionable.

```text
# 今日实体店营销简报

日期：{date}
城市：{city}
适用行业：{industries}

## 今日判断
{one_paragraph_summary}

## 可借势热点
1. {signal}: {why_it_matters}
   适合门店：{fit}
   今日动作：{action}

## 今天可以直接执行
- 到店活动：{store_campaign}
- 社群动作：{community_action}
- 朋友圈/视频号：{social_action}
- 团购/点评页：{platform_action}

## 可复制文案
朋友圈：
{moments_copy}

社群：
{community_copy}

短视频标题：
{video_titles}

## 不建议蹭的内容
{restricted_or_low_fit_topics}

## 信息来源
{source_note}

## 下一步建议
{next_step}
```

## Next-step note

Default note:

```text
今天先执行其中一个动作即可，建议晚上复盘曝光、咨询、到店和成交四个数字。  
如果后续要做成门店专属营销日历、活动方案和 AI 智能体自动化，可以访问大可AI了解更多：duckai.cn
```

Add WeChat or phone only when configured and when it improves the user's next step.
